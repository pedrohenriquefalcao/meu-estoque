# Meu Estoque

API de gestão de estoque desenvolvida em Java.

## 📋 Descrição

Projeto educacional para gerenciar produtos e controlar inventário através de uma API REST. 

**Desenvolvido apenas com Java Core**, com o objetivo de aprender e consolidar os fundamentos da linguagem Java, incluindo conceitos como orientação a objetos, manipulação de arquivos e requisições HTTP.

## 🛠️ Tecnologias

- Java
- Maven

## 📦 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/pedrohenriquefalcao/meu-estoque
cd meu-estoque-java
```

2. Compile o projeto:
```bash
mvn clean install
```

## 🚀 Como Usar

```bash
mvn run
```
